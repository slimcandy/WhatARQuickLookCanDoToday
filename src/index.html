What AR Quick Look can do today?

Apple is rumoured to reveal augmented reality glasses next year. Already, iOS devices, one of the few, support AR (augmented reality) technology on websites.

Syntax for augmented reality objects requires a link with a rel attribute with the value of ar and a picture inside.

<a rel="ar" href="/assets/models/my-model.usdz"> <img src="/assets/models/my-model-thumbnail.jpg" /> </a>

An additional field for the bottom of the view screen allows you to specify the characteristics of the virtual object.

Halloween Masks

  1. Get .usdz, .obj, .gltf or .usd file.

  2. Download Apple Reality Converter from Apple website. Drop mask file and convert to .usdz - File → Export.

  3. Send file to iOS device with AirDrop for example.

  4. Open Apple Reality Composer and import the file.

    1. Start a new Project, choose Face and attach mask to face.

    2. Click share button to export file.

    3. This .usdz file can be sent via iMessage, mail and embedded in Web.
